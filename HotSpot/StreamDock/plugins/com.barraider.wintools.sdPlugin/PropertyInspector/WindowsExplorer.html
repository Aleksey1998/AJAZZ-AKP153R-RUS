Для получения инструкций по использованию этого действия НАЖМИТЕ ЗДЕСЬ

1. Откройте проводник Windows и перейдите в каталог, который вы хотите сохранить

2. Пока проводник является активным окном, нажмите и удерживайте эту клавишу Stream Deck, пока не отобразится имя каталога

3. Теперь нажимайте коротко каждый раз, когда вы хотите увидеть этот каталог

4. Повторяйте шаги 1–2 каждый раз, когда вы хотите изменить сохраненный каталог

Действия при коротком нажатии
